# Python-projects

1.Coin Flip Game

A command-line Coin Flip Simulator built with Python. Supports single flips with animation, multi-flip statistical analysis with bar charts, and a streak guessing challenge. Built using only Python's built-in modules — no installations required. A beginner-friendly project demonstrating use of random, time, and os modules.

2.Number Guessing Game

A fun terminal-based Number Guessing Game built with Python.
This project includes multiple game modes with hints, countdown timers, and challenge modes to make the game more interactive.

3.Password Manager 

A simple and secure command-line Password Manager built with Python. Create your account, store passwords for different websites, view them anytime, and delete them when needed — all protected behind your personal login.
